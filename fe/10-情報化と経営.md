# 情報化と経営

## 10-01 関連法規

**知的財産(知的所有権)**  
知的財産は、知的創造物に対する権利であり、著作権や産業財産権がある。  

* 著作権  
著作物を創作した時点で発生。プログラム言語や規約、解放（アルゴリズム）は保護対象外

* 産業財産権  

| 名称 | 概要 |
| - | - |
| 特許権 | 高度な発明を保護 |
| 実用新案権 | 改良などの考案を保護 |
| 意匠権 | 製品のデザインを保護 |
| 商標権 | 商品名やマークなどの商標を保護 |

* 製造物責任法  
製造物の結果により、人の生命、身体などにかかる被害が生じた場合、過失の有無に関わらず製造業者等の損害賠償の責任について定める。PJ法ともよぶ。

* 労働基準法  
最低賃金、労働時間、休息などの労働状況において最低限守らなければならないこと

* 不正競争防止法  
秘密として管理されている事業活動に有用な技術上又は営業上の情報を保護する

* 不正アクセス禁止法  
他人のIDやパスワードなどを無断で使用する行為、セキュリティホールを攻撃してコンピュータに侵入する行為などの不正行為を禁止する

* 個人情報保護法  
個人情報の不適切な取扱いにおいて個人の権利利益が侵害されることを未然に防ぐ

**派遣と請負**  

* 派遣  
労働者の就業条件の中で、雇用関係と指揮命令権が切り離されている形態

* 請負  
受託者が委託者から業務を請け負う、瑕疵担保責任を伴う場合がほとんど。

## 10-02 企業会計

**管理会計**  
企業内部の意思決定や組織統制を目的とする

**財務会計**  
企業の経営者が株主や債権者などの企業外部の利害関係者(ステークホルダ)に対して会計報告を行うことを目的とする

**費用と利益**  
利益 ＝ 売上高 - 総費用 = 売上高 - (固定費＋変動費)

**損益分岐点**  
損益分岐点 = 固定費 / (1-変動比率)

**在庫評価**  
| 名称 | 概要 |
| 先入先出法 | 先に仕入れた商品から先に売れたものとみなす |
| 後入先出法 | 後に仕入れた商品から先に売れたものとみなす |
| 移動平均法 | 商品を購入した都度、在庫金額と購入価額との合計を、在庫数量と購入数の合計で割り、払出単価を算出する |
| 総平均法 | 初期在庫の評価額と仕入れた商品の総額の合計をその総数量で割り、払出単価を算出する |

**貸借対照表**  
会計期間末日時点のすべての資産、負債、純資産などを記載したもの。B/S(Balance Sheet)とも呼ばれる。  
ある時点の資産・負債・純資産


**損益計算書**  
会計期間に属するすべての収益と費用を記載し、算出した利益を示したもの。  
P/L(Profit and Loss statment)。  
ある期間の収益と費用

## 10-03 経営戦略と技術戦略

**経営やマーケティングに関する用語**

* ベンチマーキング  
最強の相手と比較する  
競合、または先進企業と比較して、製品、サービスおよびオペレーションなどを訂正的、定量的に把握する

* コアコンピタンス  
核となるノウハウや技術に集中する  
他社にまねできない独自のノウハウや技術力などに経営資源を集中し、競争優位を確立する

* M&A(Merger and Acquisition)  
企業の合併と買収  
新規分野への進出や、事業の拡大、再編などのために他社の経営資源を獲得する  

* アライアンス  
企業提携  
新しいビジネスモデルや技術を開発するために、各企業が取引関係や競合関係、又は業界を超えて提携する

* プロダクトポートフォリオ  
PPM(Product Portfolio Management)は、ビジネスを4つの事象に分類し、経営資源配分のバランスをみるマネジメント手法

| 名称 | 概要 |
| - | - |
| 問題児 | 事業としての魅力があり、資金投下を行えば将来の資金供給源になる可能性がある |
| 花形 | 現在は大きな資金の流入をもたらしているが、同時に将来にわたっても資金の投下も必要 |
| 金のなる木 | 現在は資金の主たる供給源の役割を果たしており、新たに資金を投下すべきではない |
| 負け犬 | 事業を継続させていくための資金投下の必要性は低く、将来的には撤退を考えざるを得ない |

* 競争戦略  

| 企業の地位 | 戦略 |
| - | - | 
| リーダ | 総市場規模を拡大することでシェアを維持しながら新規需要の獲得を図る |
| チャレンジャ | 市場での地位向上とトップシェア奪取を目的とした差別化戦略の展開を図る |
| ニッチャ | 他社が参入しにくい特定の市場に対して専門化し、高利益率を図る |
| フォロワ | リーダの行動を観察し、迅速に模倣することで製品開発などのコスト削減を図る |

* SWOT分析  
強み(Strength)、弱み(weekness)、機会(oppotunity)、脅威(threat)

* マーケティングミクス  
製品（Product)、価格（Price)、流通（Place)、プロモーション（Promotion)を組み合わせる

* プロダクトライフサイクル  
製品の 導入期 → 成長期 → 成熟期 → 衰退期 を表す

* バリューチェーン分析  
企業の事業活動を機能ごとに主活動と支援活動にわけ、企業が顧客に提供する製品やサービスの利益はどの活動で生み出されているかを分析する

* ワンツーワンマーケティング
個々の顧客ニーズに対応する

* マスカスタマイゼーション  
大量生産、大量販売のメリットを生かしつつ、きめ細やかな仕様・機能の取り込みによって顧客一人一人の好みに応じられる製品やサービスを提供する

* バランススコアカード  
企業のビジョンと戦略を実現するために財務、顧客、業務プロセス、学習、成長の4つの視点から評価指標を設定する

* ナレッジマネジメント  
個々の社員がビジネス活動から得た客観的な知識や経験、ノウハウなどを企業全体の知識として共有する

* CRM  
Customer Relationship Management  
ITを利用して顧客に関する情報を収集、分析し、長期的視点から顧客と良好な関係を築く

* SFA  
Sales Force Automation  
営業課都度にITを活用して営業効率を品質管理を高め、売り上げ、利益の大幅な増加や顧客満足度の向上を図る

* SCM
Suppuly Chain Management  
部品の調達から製造、流通、販売に至る一連のプロセスに参加する部門と企業間で情報を共有、管理する

* ERP  
Enterprise Resource Plannning  
企業全体の経営資源を有効かつ総合的に計画して管理し、経営の効率向上を図る

**技術戦略に関する用語**

* MOT
Management of Technology  
技術力をコアコンピタンスとする企業が、事業を持続的に発展させていくために技術開発に投資してイノベーションを促進していくこと  

| 名称 | 概要 |
| - | - |
| プロダクトイノベーション | 革新的な新製品を開発するといった、製品そのものに対する技術革新 |
| インクリメンタルイノベーション | 既存製品の細かな部分改良を積み重ねる技術革新 |
| ラディカルイノベーション | 経営構造の全面的な改革を必要とする技術革新 | 
| プロセスイノベーション | 研究開発過程、製造工程、および物流家庭の技術革新 |

* コンカレントエンジニアリング  
Concurrent Enginnering  
各工程作業を並行に行うことで、スピードアップやコストダウンを図る  

**生産戦略に関する用語**  

* ジャストインタイム  
JIT  
必要なものを、必要なときに、必要なだけ生産する

* セル生産方式  
部品の組み立てから完成検査までの全工程を一人または数人程度で作業する生産方式

* MRP  
Materials Requirements Planning  
製品を生産するために必要となる部品や資材の量を計算し、生産計画に反映させる資材管理手法  

* CAD  
Computer Aided Manufacturing  
コンピュータを使って製造作業を支援すること

**経営組織**  

| 名称 | 概要 |
| - | - |
| 職能別組織 | 購買・生産・販売・財務などの仕事の性質によって部門を構成した組織 |
| ラインアンドスタッフ組織 | 事業を遂行するピラミッド型のラインと、業務を専門的立場からサポートするスタッフからなる組織。 |
| 事業部制組織 | 利益責任と業務遂行に必要な職能を、製品別、顧客または地域別にもつことによって、自己完結的な経営活動が展開できる組織である |
| マトリックス組織 | 自己の専門とする職能部門と特定の事業を遂行する部門の両方に所属する組織 | 
| プロジェクト組織 | 特定の課題のもとに各部門から専門家を集めて編成し、期間と目標を定めて活動する一時的かつ柔軟な組織 | 
| 社内ベンチャー | 新事業の開発のために社内に独立した活動単位として設置し、小さな企業であるかのように運営する組織 |

## 10-04 ビジネスインダストリ

**EC(Electronic Commerce)**  
消費者向けや企業間の商取引をインターネットなどの電子ネットワークを活用して行うこと

| 分類 | 概要 |
| - | - |
| CtoC | Consumer to Consumer 個人間取引 |
| BtoC | Businnes to Consumer 企業対個人取引 |
| BtoB | Business to Business 企業間取引 |
| BtoE | Business to Employee 企業対従業員取引 |
| GtoC | Government to Citizen 政府対個人取引 |
| GtoB | Government to Business 政府対企業取引 |
| GtoG | Government to Government 政府間取引 |

**ロングテール**  
通常の店舗では売り場の問題から売れ筋商品しか並べられないが、  
インターネットショッピングではあまり売れない商品も置き続けることができ、  
この売り上げや利益が無視できないぐらい大きいことをいう

**カードシステム**  
磁気カードやICカード

| 種類 | 概要 |
| - | - |
| デビットカード | 買い物代金の支払いを銀行のキャッシュカードで行い、利用金額を預金口座から即座に引き落とす |
| クレジットカード | カード申込者に対して、信用限度額内の買い物を代金後払いで行う |
| プリペイドカード | 商品購入を前提に代金を前払いしてもらい、商品購入の都度、カードから減額していく |
| ポイントカード | 購入額に対応した点数をカードに累積し、累積高に応じてサービスを受ける |

**ソリューションビジネス**  

* ASP(Application Service Provider)
* SaaS(Software as a Service)  
インターネット経由でアプリケーションを配信するサービス

* アウトソーシング  
情報システムの開発や運用保守に関わる全部又はほとんどの機能を外部の専門企業に委託する形態

* ホスティングサービス  
サービス提供事業者が、利用者に自社の建物内に設置したサーバや通信機器を貸し出すサービス

* ハウジングサービス  
サービス提供事業者が、利用者の通信機器やサーバを自社の建物内に設置して運用するサービス

* オンラインストレージ  
サーバ上のディスクスペースをファイル保管用に貸し出すサービス

**クラウドコンピューティング**  
インターネットを経由してハードウェアやソフトウェア、データベースなどのコンピュータの各種資源を利用するサービスの総称
