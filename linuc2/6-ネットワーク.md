# ネットワーク

## ネットワークの設定

### ネットワークデバイス

*ネットワークインタフェイス*
| インタフェイス名 | 説明 |
| - | - |
| eth0 | 1番目のイーサネットインタフェイス |
| eth1 | 2番目のイーサネットインタフェイス |
| ppp0 | 1番目のPPPインタフェイス |
| lo | ローカルループバックインタフェイス |

*インタフェイスの命名規則*
| 種別 | 説明 |
| - | - |
| en | イーサネット |
| wl | 無線LAN |
| ww | 無線WAN |

| デバイス | 説明 |
| - | - |
| オンボードデバイス | o<インデックス> |
| ホットプラグデバイス | s<スロット> |
| PCIデバイス | p<バス>s<スロット> |
